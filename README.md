# ProjektniZadatak

## Create **venv**
### Windows
    python -m venv venv
### Linux/MacOS
    python3 -m venv venv

## Activate **venv** 
### Windows
    venv\Scripts\activate
### Linux/MacOS
    source venv/bin/activate

## Install requirements
    pip install -r requirements.txt

## Install MySql
    pip install flask-mysqldb

## Install Flask Bootstrap
    pip install flask-bootstrap

## Install passlib
    pip install passlib

## Set FLASK_APP
    set FLASK_APP=app.py

## Run app 
    flask run
